# Samples

This directory contains example projects to demonstrate how `dock-docs` works.

Each subdirectory contains:
- A `Dockerfile` (with Magic Comments).
- A `dock-docs.yaml` configuration file.
- A `README.md` generated by the tool.

## Available Samples

### [Python](python/)
Demonstrates:
- Documentation of `ENV` variables with default values.
- Comparison of different Python base images (`3.9-slim`, `3.10-slim`, `3.11-slim`).

### [Node.js](node/)
Demonstrates:
- Documentation of `ARG` build arguments.
- Comparison of different Node.js Alpine versions (`16`, `18`, `20`).

### [Go](go/)
Demonstrates:
- Multi-stage build documentation.
- Comparison of different Go Alpine versions.

## How to Run

To regenerate the documentation for a sample, navigate to its directory and run `dock-docs`:

```bash
cd samples/python
dock-docs --config dock-docs.yaml
```
